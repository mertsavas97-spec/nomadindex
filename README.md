# NomadIndex

NomadIndex is an SEO-first web platform designed to help digital nomads compare cities around the world based on cost of living, internet quality, safety, weather, visa information, and other lifestyle factors.

Built with **Next.js**, **TypeScript**, and **Vercel**, the project emphasizes performance, scalability, and modern web architecture while delivering a fast and intuitive user experience.

## Live Demo

🌐 https://nomadindex.app

## Features

* Global city directory
* City comparison tools
* Cost of living insights
* Visa and lifestyle information
* Advanced filtering and search
* SEO-optimized pages
* Dynamic metadata and Open Graph support
* XML sitemap and robots.txt
* Responsive design
* Optimized performance and accessibility

## Tech Stack

* Next.js (App Router)
* React
* TypeScript
* Tailwind CSS
* Vercel

## Project Goals

* Help users discover the best cities for remote work and long-term travel
* Deliver fast, SEO-friendly pages with excellent Core Web Vitals
* Build a scalable platform ready for future data expansion
* Provide a clean, intuitive experience across desktop and mobile devices
